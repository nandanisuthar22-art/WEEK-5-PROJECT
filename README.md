# WEEK-5-PROJECT
# 🩺 Non-Alcoholic Fatty Liver Disease (NAFLD) Prediction

## 📌 Overview

This project focuses on predicting **Non-Alcoholic Fatty Liver Disease (NAFLD)** using machine learning techniques. NAFLD is a growing global health concern, and early detection can significantly improve patient outcomes.

The notebook demonstrates the full pipeline from data preprocessing to model evaluation, helping in understanding patterns and risk factors associated with NAFLD.

---

## 🚀 Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection and engineering
* Machine Learning model implementation
* Model evaluation and performance comparison
